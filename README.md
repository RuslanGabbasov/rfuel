# Бензин-бот

Telegram-бот для мониторинга наличия топлива на ближайших заправках через API [gdebenz.ru](https://gdebenz.ru).

## Возможности

- 📍 Отслеживание заправок в радиусе 10 км от указанной геопозиции
- ⛽ Поддержка всех типов топлива: АИ-92, АИ-95, АИ-98, АИ-100, ДТ, Газ
- 🔔 Уведомления при появлении нужного топлива поблизости
- 📊 Информация о статусе заправки, очереди и количестве подтверждений
- ⏱ Проверка каждые 2 минуты

## Установка и запуск

```bash
# 1. Клонировать репозиторий
cd benz

# 2. Создать виртуальное окружение
python3 -m venv venv
source venv/bin/activate

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Запустить бота
python bot.py
```

## Использование

1. Найти бота в Telegram: `@rfuel_bot`
2. Отправить `/start`
3. Отправить свою геопозицию
4. Выбрать тип топлива
5. Бот начнёт мониторинг и пришлёт уведомление, когда топливо появится

### Команды

| Команда | Описание |
|---------|----------|
| `/start` | Начать настройку |
| `/help` | Справка |
| `/status` | Текущие настройки |
| `/setlocation` | Изменить локацию |
| `/setfuel` | Изменить тип топлива |
| `/stop` | Остановить мониторинг |
| `/startmon` | Запустить мониторинг |

## Принцип работы

1. Бот сохраняет координаты и тип топлива пользователя в SQLite
2. Каждые 2 минуты фоновый job запрашивает API `gdebenz.ru/api/nearby` для всех активных пользователей
3. Для каждой заправки со статусом `yes` или `queue` проверяется наличие нужного топлива в поле `detail`
4. Если топливо найдено и пользователь ещё не был уведомлён об этой заправке — отправляется сообщение
5. Когда топливо на заправке заканчивается, запись об уведомлении удаляется, чтобы можно было уведомить снова при следующем появлении

## Данные API

Данные предоставлены сервисом [gdebenz.ru](https://gdebenz.ru). Бот использует:

- `GET /api/nearby?lat=...&lon=...&radius_km=...` — список заправок
- `GET /api/comments/{osm_id}/recent?limit=12` — последние отметки (опционально)
